Nintendo Stock Data Dashboard Trends


About the dashboard project:

Welcome to the Nintendo Stock market analysis dashboard.In this project, I have built an interactive and dynamic web application that provides a comprehensive view of the stock’s price movements and market activity for Nintendo to help internal and external stakeholders including buyers and sellers to analyze market sentiment.The dataset contains detailed historical stock price data for Nintendo, covering the period from 1996-11-18 to 2024-06-14.The data is collected from a public web site kaggle.com
and includes daily records of the stock's opening price, highest price, lowest price, closing price, and trading volume.
